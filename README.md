# Cheonan Travel Video Manager

Менеджер туристических видеозаписей города Чхонан.

## Технологии
- React + Vite
- Zustand (state management)
- @dnd-kit (drag & drop)

## Запуск

```bash
npm install
npm run dev
```

## Возможности

- Загрузка нескольких видео (клавиша `F`)
- Навигация между видео (`<` / `>`)
- Временная шкала с drag-and-drop
- Автоматические снимки кадров (5 / 10 / 15)
- Установка точек In/Out (`I` / `O` / `R`)
- Удаление видео (`D`)
- Галерея с миниатюрами и сортировкой (Drag & Drop)
- Предварительный просмотр смонтированных сегментов
- Сохранение состояния в localStorage
